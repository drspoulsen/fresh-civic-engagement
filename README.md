# Longitudinal_Civic_Eng
Dr. Clarke De Reza and Dr. Grosse's longitudinal data

Campus Civic Engagement Data – Data Analytics Intern Job Description 
Goal: Merge three years of campus survey data on student civic engagement into one longitudinal data set in an Excel/CSV format 

-or-

Goal: Merge repeated measures/longitudinal variables from three years of student surveys on civic engagement into one data set in Excel/CSV format. 
Tasks:
1)	Create a new variable that identifies year of survey completion/indicates the data set source (e.g. SP23, SP24, SP25)
2)	Identify common variables across years and standardize variable names for consistency 
3)	Recode categorical variables consistently across set
4)	Recode qualitative/open-response variables (e.g. major/minor) into standard categorical variables using college course catalog tags (e.g. MAT, MFCE, CMS)
5)	Align measurement scales so low numbers always relate to disagreement and high variables always relate to agreement 
6)	Identify missing variables across data sets and re-coding empty cells “-9” so they can be designated as discrete missing values in SPSS analysis 
7)	Write a codebook describing variables and recoding process
